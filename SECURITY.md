# Security Policy | Cyber-Vault v9.9.9

This security policy defines the vulnerability reporting protocols for the **Cyber-Vault** module.

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 9.9.9   | :white_check_mark: |
| < 9.0   | :x:                |

## Reporting a Vulnerability
If you identify a security flaw within this encryption suite, please do not disclose it publicly via GitHub Issues. Instead, contact the developer privately through the contact information provided on the GitHub profile.

I aim to address critical security concerns within 48-72 hours to ensure the academic and professional integrity of this BSc in CSE portfolio project.
